ProductsMeasures

ERDEntity
MeimBoxDB :: Model :: Data Model1 :: ProductsMeasures

Description

documentation

Cette table regroupe les dimensions d'un boxproduct pour une taille donnée

contraintes

Le couple prodId-sizename forme la clef étrangère [prodId-sizename]=>ON DELETE CASCADE, ON UPDATE CASCADE

⚠️Chaque boxproduit doit avoir ses dimension encodé dans le système ⚠️Il faut vérifier que chaque dimension du client soit inférieur ou égale au dimension du produit qu'il désire (en effet on ne peut pas augmeneter la taille des produits)
