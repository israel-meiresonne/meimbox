Definition

Cette table regroupe les traduction disponibles.

À chque nouvelle traduction disponible on ajoute une colonne à cette table.

Definition: v2.0

Cette table regroupe les traductions dynamique cad qu'il permet d'implémenter une map qui portera comme clef un texte dans le language par default du système(anglais) et chaque clef pointera vers une traduction dans la langue choisis. ex: $trad["i'm angry"]["fr"] => j'ai faim
